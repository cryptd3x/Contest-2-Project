# Tetris - MASM x86 Assembly Game

A classic Tetris clone written in x86 Assembly Language using MASM (Microsoft Macro Assembler).

By: Justin Bower

## How to Build and Run the Game

### Step 1: Install Visual Studio 2022 (if not already installed)

1. Download and install **Visual Studio 2019 CLI Dev Environment* using winget:  
   [winget install --id Microsoft.VisualStudio.2019.BuildTools -e]

### Step 2: Open the x86 Native Tools Command Prompt

1. Click the **Windows Start button**.
2. Type: `Developer Command Prompt for VS 2019`
3. Launch **"Developer Command Prompt for VS 2019"**  

### Step 3: Navigate to the Project Folder

In the command prompt, and 'cd' into to your project directory:

```cmd
cd C:\Path\To\Your\Tetris\Project```

This will create a .exe file for running the game.

### Step 4: Run the .bat file
Run the batch file in the Visual Studio 2019 CLI Dev Environment.,

Use the arrow keys as controls.

Enjoy the game :D
